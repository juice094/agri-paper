# 目标期刊与投稿策略

## 主投目标

### 1. 中文核心（保底）
- **期刊**：《农业工程学报》或《农业图书情报学报》
- **类型**：中文核心期刊（北大核心）
- **审稿周期**：1–2 个月
- **优势**：
  - 中文写作，课程作业可直接转化
  - 对方法论类论文友好
  - 审稿快，可作为课程答辩基础
- **投稿时机**：W5 完成中文稿后直接投，确保课程学分

### 2. SCI 三区（主投）
- **期刊**：*Computers and Electronics in Agriculture* (COMPAG)
- **ISSN**：0168-1699
- **影响因子**：~8.3 (2024)
- **分区**：JCR Q1 / 中科院农林科学 1 区 Top（部分年份 Q2，按最新版确认）
- **审稿周期**：2–4 个月
- **收稿方向**：
  - 农业信息化、精准农业
  - AI 与决策支持系统在农业中的应用
  - 知识管理与农业专家系统
- **论文类型**：Research Article（6,000–8,000 words）
- **LaTeX 模板**：Elsevier 官方 `elsarticle.cls`
- **投稿时机**：W6 润色后通过 Editorial Manager 系统投稿
- **创新性要求**：
  - 必须明确声明 Novelty Statement（方法创新性 or 应用场景首发性）
  - 强调"配置驱动"（Declarative Config）与"本地优先"（Local-First）的方法论贡献

### 3. 预印本（占坑）
- **平台**：arXiv cs.AI / AgriRxiv
- **目的**：
  - W3 实验有初步结果后即可上传，建立优先权日期戳
  - 防止研究成果在审稿期间被抢发
- **内容**：可上传完整 manuscript v1 或技术报告摘要

---

## 投稿路径建议

```
W3  ──→ 上传 arXiv 预印本（建立优先权）
W5  ──→ 投稿《农业工程学报》（中文核心，课程作业）
W6  ──→ 投稿 Computers and Electronics in Agriculture（SCI，主目标）
```

## 格式要求速查

| 项目 | COMPAG (Elsevier) | 农业工程学报 |
|------|-------------------|--------------|
| 字数 | 6,000–8,000 words | 8,000–10,000 字 |
| 摘要 | 250 words, structured | 300–400 字，结构式 |
| 关键词 | 4–6 个 | 3–8 个 |
| 图表 | 矢量图优先，≥300 dpi | 清晰即可，黑白印刷需考虑 |
| 参考文献 | ≥30 篇，近 5 年占 60% | ≥20 篇，近 5 年占 50% |
| 代码/数据 | 鼓励提供 GitHub + DOI | 可选 |
| 查重 | <15% | <20% |
